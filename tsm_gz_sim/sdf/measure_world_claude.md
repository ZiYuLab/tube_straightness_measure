# 1. 基本信息

* gazebo 版本： Homonic
* ros2 版本： jazzy
* sdf类型： 世界
* 功能： 构建管道直线度测量的检装置

# 2. 构成

## 1. 管道模型部分

由于管道类型多样，因此模型应该由launch的时候从外部加载，因此此处并不载入模型而是预留位置。模型位于x轴正中心，管道轴向为x轴，管道中心位于 z=0。

## 2. 检测相机部分

采用RGBD相机两个，从侧上方 45° 向内俯视管道，需要能够看清钢管局部范围，并通过点云拼接至少能看到半截钢管，位于yz平面上，在管道正上方。
为实现任务需要添加相应的传感器插件，并添加用于可视化传感器位置的几何体。传感器不会移动，因此请设置为静态。

# 3. 当前实现状态（measure_world.sdf）

## 相机布局

| 模型名 | 位置 (x, y, z) | 姿态 rpy | 话题前缀 |
|--------|---------------|----------|---------|
| rgbd_1 | 0, +1.0, 1.0 | 0, π/4, +π/2 | /rgbd_1 |
| rgbd_2 | 0, -1.0, 1.0 | 0, π/4, -π/2 | /rgbd_2 |

- 相机默认光轴为 -Z（光学坐标系），pitch=π/4 使光轴向下 45°，yaw=±π/2 使光轴朝向管道中心
- 两台相机从 ±Y 侧各以 45° 斜向下看管道截面，点云拼接覆盖管道上半截面
- 无地面（管道中心在 z=0）

## 话题

每台相机发布：
- `/rgbd_1/image`、`/rgbd_1/depth_image`、`/rgbd_1/points`
- `/rgbd_2/image`、`/rgbd_2/depth_image`、`/rgbd_2/points`

## 相机参数

- 分辨率：640×480
- 水平视角：60°（1.047 rad）
- 深度范围：0.1m ~ 5.0m
- 帧率：10 Hz

## 管道加载方式

```bash
ros2 run ros_gz_sim create -world measure_world -file straight_tube.sdf
```
